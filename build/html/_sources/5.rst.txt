5 外观观测
===================

.. raw:: html

  <h2 id="test111">5 外观观测</h2>

.. raw:: html

 <p style="text-align:justify"><a href="#id5.0.1"> 5.0.1</a> <span id="id5.0.1"> 外观观测可分为裂缝观测、构件表面变形观测、结构缝观测、表面破损观测、结构防腐层破损观测等。外观观测应符合下列规定。</span></p>
 <p style="text-align:justify"><a href="#id5.0.1.1"> 5.0.1.1</a> <span id="id5.0.1.1"> 裂缝观测应包括裂缝的分布位置、走向、长度、宽度、深度和变化过程等内容。</span></p>
 <p style="text-align:justify"><a href="#id5.0.1.2"> 5.0.1.2</a> <span id="id5.0.1.2"> 构件表面变形观测应包括变形范围、变形程度和变形方向等内容。</span></p>
 <p style="text-align:justify"><a href="#id5.0.1.3"> 5.0.1.3</a> <span id="id5.0.1.3"> 结构缝观测应包括缝宽变化、结构缝两侧结构的相对错动情况等内容。</span></p>
 <p style="text-align:justify"><a href="#id5.0.1.4"> 5.0.1.4</a> <span id="id5.0.1.4"> 表面破损观测应包括破损部位及分布状况、破损形态和破损的几何尺寸。</span></p>
 <p style="text-align:justify"><a href="#id5.0.1.5"> 5.0.1.5</a> <span id="id5.0.1.5"> 结构防腐层破损观测应包括破损部位、破损分布状况、破损面积等内容。</span></p>
 <p style="text-align:justify"><a href="#id5.0.2"> 5.0.2</a> <span id="id5.0.2"> 外观观测点的设置应符合下列规定。</span></p>
 <p style="text-align:justify"><a href="#id5.0.2.1"> 5.0.2.1</a> <span id="id5.0.2.1"> 裂缝观测点的数量和设置应根据观测目的和要求确定。观测区及观测点应能确切反映结构或构件开裂程度、裂缝特征及变化趋势。每条裂缝的测试区不宜少于3个，每个测试区内裂缝宽度观测点不宜少于4个。</span></p>
 <p style="text-align:justify"><a href="#id5.0.2.2"> 5.0.2.2</a> <span id="id5.0.2.2"> 构件表面变形观测点的设置应能确切反映构件的实际变形范围、变形程度和变形趋势，并应满足下列要求：</span></p>
 
 <ol>
 <li>构件边角处或构件表面变形区以外，观测点不少于3个；</li>
 <li>构件表面变形边缘处，观测点不少于4个；</li>
 <li>构件表面变形区及敏感部位，观测点不少于5个，并至少形成2个测试断面。</li>
 </ol>

 <p style="text-align:justify"><a href="#id5.0.2.3"> 5.0.2.3</a> <span id="id5.0.2.3"> 结构缝宽观测点应均匀设置在结构缝的两端和中间部位，观测点数量不得少于3个。</span></p>
 <p style="text-align:justify"><a href="#id5.0.2.4"> 5.0.2.4</a> <span id="id5.0.2.4"> 表面破损观测点应根据结构构件所处的位置分区域设置，并应重点设置在浪溅区和水位变动区。</span></p>
 <p style="text-align:justify"><a href="#id5.0.2.5"> 5.0.2.5</a> <span id="id5.0.2.5"> 结构防腐层破损观测点应根据构件所处的位置分区域设置，并应重点设置在浪溅区和水位变动区。</span></p> 
 
 <p style="text-align:justify"><a href="#id5.0.3"> 5.0.3</a> <span id="id5.0.3"> 裂缝和构件表面变形观测可采用下列方法。</span></p>

 <p style="text-align:justify"><a href="#id5.0.4"> 5.0.4</a> <span id="id5.0.4"> 裂缝观测周期应根据裂缝变化速度确定。当裂缝加大或发生异常情况时，应加密观测。</span></p>

 <p style="text-align:justify"><a href="#id5.0.5"> 5.0.5</a> <span id="id5.0.5"> 裂缝宽度观测精度不应低于0.05mm,裂缝长度和深度观测的精度不宜低于I mm;构件表面变形的观测精度应符合《水运工程测量规范》(JTS 131)的有关规定；结构缝的观测精度不应低于0.5mm。</span></p>

 <p style="text-align:justify"><a href="#id5.0.6"> 5.0.6</a> <span id="id5.0.6"> 外观观测应提交下列成果：</span></p>
 <ol>
 <li>观测说明；</li>
 <li>观测点布置图；</li>
 <li>外观观测记录表和影像资料；</li>
 <li>裂缝分布图、裂缝观测记录表、裂缝发展过程图、裂缝与其他相关因素的关系资料，表面变形曲线图，结构缝变化过程线图，破损分布图，裂缝成因分析等；</li>
 <li>外观观测成果分析。</li>
 </ol>
