3 基本规定
===============

.. raw:: html

  <h2 id="test111">3 基本规定</h2>

3.1 一般规定
---------------------------------
.. raw:: html

 <p style="text-align:justify"><a href="#id3.1.1"> 3.1.1</a> <span id="id3.1.1"> 运工程水工建筑物原型观测可分为外观观测、变位与变形观测、力与应力观测、振动观测、冻融观测和腐蚀观测。</span></p>
 <p style="text-align:justify"><a href="#id3.1.2"> 3.1.2</a> <span id="id3.1.2"> 水运工程水工建筑物原型观测项目应按表3.1.2选定。</span></p>
 <ol>
 <li style="text-align:justify">影响外观、使用舒适性或结构使用功能的变形；</li>
 <li style="text-align:justify">造成人员不舒适或结构使用功能受限的振动；</li>
 <li style="text-align:justify">影响外观、耐久性或结构使用功能的局部损坏。</li>
  </ol>
 <p style="text-align:justify"><a href="#id3.1.3"> 3.1.3</a> <span id="id3.1.3"> 观测项目的选定应考虑全面、突出重点。对观测项目影响较大的环境因素应同步进行观测。</span></p>
 <p style="text-align:justify"><a href="#id3.1.4"> 3.1.4</a> <span id="id3.1.4"> 水运工程水工建筑物原型观测应符合下列规定。</span></p>
 <p style="text-align:justify"><a href="#id3.1.4.1"> 3.1.4.1</a> <span id="id3.1.4.1"> 观测方法、观测设备的选用和观测点的布置应根据观测要求、建筑物结构型式和所处的自然环境确定，并能全面反映建筑物的工作状态。</span></p>
 <p style="text-align:justify"><a href="#id3.1.4.2"> 3.1.4.2</a> <span id="id3.1.4.2"> 观测周期应根据观测目的和要求确定，并应能反映观测要素的变化过程；出现异常情况时，应增加观测频次。</span></p>
 <p style="text-align:justify"><a href="#id3.1.4.3"> 3.1.4.3</a> <span id="id3.1.4.3"> 观测精度应满足观测设计要求。</span></p>
 <p style="text-align:justify"><a href="#id3.1.4.4"> 3.1.4.4</a> <span id="id3.1.4.4"> 观测作业应保持人员、仪器、时间和观测环境的相对固定和观测资料的连续性。</span></p>
 <p style="text-align:justify"><a href="#id3.1.4.5"> 3.1.4.5</a> <span id="id3.1.4.5"> 观测记录应书写规范、字迹清楚、填写齐全，不得更改。修改笔误必须杠改，不得涂改。记录的数据应及时进行计算和校核。</span></p>
 <p style="text-align:justify"><a href="#id3.1.5"> 3.1.5</a> <span id="id3.1.5"> 水运工程水工建筑物原型观测工作应包括下列内容：</span></p>
  <ol>
 <li style="text-align:justify">观测设计；</li>
 <li style="text-align:justify">观测仪器设备选型、安装、使用和维护；</li>
 <li style="text-align:justify">必要的观测辅助设施布置；</li>
 <li style="text-align:justify">观测实施；</li>
 <li style="text-align:justify">观测资料整理与分析；</li>
 <li style="text-align:justify">观测成果报告编写。</li>
  </ol>
 <p style="text-align:justify"><a href="#id3.1.6"> 3.1.6</a> <span id="id3.1.6"> 原型观测的项目合同、技术资料、观测成果及分析计算资料、观测成果报告和审查鉴定意见书等应按国家科技档案规定和档案管理要求整理归档。</span></p>
 <p style="text-align:justify"><a href="#id3.1.7"> 3.1.7</a> <span id="id3.1.7"> 观测单位应具有相应资质，所使用的观测仪器应满足国家计量法规和标准的要求。</span></p>



3.2 观测仪器设备
----------------------------------

.. raw:: html

 <p style="text-align:justify"><a href="#id3.2.1"> 3.2.1</a> <span id="id3.2.1"> 测仪器设备必须经检验和标定合格后才能使用，并应在检定周期内使用。</span></p>
 <p style="text-align:justify"><a href="#id3.2.2"> 3.2.2</a> <span id="id3.2.2"> 测仪器设备应满足操作方便、抗干扰能力强、稳定性和耐久性好的要求，其量程、精度指标应符合观测设计要求。观测仪器设备应定期进行维护和保养。</span></p>
 <p style="text-align:justify"><a href="#id3.2.3"> 3.2.3</a> <span id="id3.2.3"> 观测仪器设备应严格按操作规程使用。</span></p>
 <p style="text-align:justify"><a href="#id3.2.4"> 3.2.4</a> <span id="id3.2.4"> 同类观测要素的量测宜采用相同型号和规格的仪器、传感器及线缆。</span></p>
 <p style="text-align:justify"><a href="#id3.2.5"> 3.2.5</a> <span id="id3.2.5"> 观测传感器的量程、精度、耐久性和稳定性等各项性能指标应满足观测设计要求，并应按照观测设计、产品说明书的要求布置、埋设和保护传感器。</span></p>
 <p style="text-align:justify"><a href="#id3.2.6"> 3.2.6</a> <span id="id3.2.6"> 观测线缆的稳定性、耐久性等应满足观测设计要求，并与观测系统整体匹配。观测线缆应合理布置或埋设，减少与工程施工或生产作业的干扰，并应采取有效保护措施。</span></p>
 <p style="text-align:justify"><a href="#id3.2.7"> 3.2.7</a> <span id="id3.2.7"> 观测系统布设完毕后，应及时对观测系统进行调试，并绘制观测系统竣工图。</span></p>
 <p style="text-align:justify"><a href="#id3.2.8"> 3.2.8</a> <span id="id3.2.8"> 观测系统发生变动时，应建立新旧系统转换关系。</span></p>
 <p style="text-align:justify"><a href="#id3.2.9"> 3.2.9</a> <span id="id3.2.9"> 采用无线传输技术时，应根据观测要求及现场环境选择合适的传输方式，确保数据传输的稳定性和可靠性。</span></p>

3.3 观测实施
----------------------------------

.. raw:: html

 <p style="text-align:justify"><a href="#id3.3.1"> 3.3.1</a> <span id="id3.3.1"> 委托方应协调组织有关单位布设观测系统，并协调采取有效措施保护观测系统。</span></p>
 <p style="text-align:justify"><a href="#id3.3.2"> 3.3.2</a> <span id="id3.3.2"> 观测实施前应对观测系统进行检查调试，观测过程中应定期检查和维护；当观测系统出现异常情况时，应查明原因及时修复，并做好修复前后的数据处理工作，应保证观测数据完整、连续。</span></p>
 <p style="text-align:justify"><a href="#id3.3.3"> 3.3.3</a> <span id="id3.3.3"> 水工建筑物维修、加固或改建时，应对需要保留的观测系统采取有效保护措施。</span></p>
 <p style="text-align:justify"><a href="#id3.3.4"> 3.3.4</a> <span id="id3.3.4"> 观测实施过程中，发现水工建筑物有异常情况时，应及时向委托方报告。</span></p>
